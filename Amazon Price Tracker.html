Introdução¶

Nos dias atuais, a tecnologia tem transformado a maneira como consumidores e empresas lidam com compras e vendas. Desta forma, o presente trabalho visa construir um tracker de preos do site da Amazon (https://www.amazon.com/), para que o consumidor possa acompanhar quando o preo diminuir. Este sistema automatizado serve para coletar e analisar mudanas nos preos de produtos, permitindo que consumidores e empresas tomem decisesmaisacertivas.

Neste projeto iremos extrair os dados de preo dos celulares Samsung Galaxy S24 Ultra e Iphone 15 Pro Max, visando, alm de poder visualizar os preos, comparar como diferentes produtos da mesma categoria se comportam.

Descrição do Problema¶

A variao de preos uma realidade no mercado de bens de consumo. Produtos podem sofrer mudanas de preo devido a promoes, sazonalidade, flutuaes de demanda, entre outros fatores. Para consumidores, a procura pelo menor preo pode ser exaustiva e demorada. Para empresas, entender a dinmica dos preos concorrentes crucial para estratgias de pricing e competitividade. O problema, portanto, reside na necessidade de um sistema eficiente que monitore os preos de produtos no site (https://www.amazon.com.br/) e que notifique automaticamente, atravs de um link, quando um preo mais baixoencontrado.

def getPandaDfForSearchResults(searchResultsPage):
    rows=[]
    for result in searchResultsPage:
        title = result.find("span", {"class": "a-text-normal"}) # ttulo do produto
        sponsored = result.find("span", {"class": "a-color-base"}, string="Sponsored") # patrocnio do produto
        price = result.find("span", {"class": "a-price-whole"}) # preo do produto
        url = result.find("a", {"class": "a-link-normal"}) # url do produto
        
        if price:
            # Cria Linha do Dataframe
            row = [title.text, bool(sponsored), price.text, "https://amazon.com.br/" + url['href']]
            rows.append(row)
    
    # Cria Dataframe e Retorna
    df = pd.DataFrame.from_records(rows, columns=["Title", "Sponsored", "Price", "URL"])
    
    return df

def getSearchResultsForPage(url):
    wd.get(url) # abre link de procura
    
    nextPageCssClasses = [] # lista para checar prxima pgina
    nextPageLink = wd.find_element(By.CLASS_NAME, "s-pagination-next") # procura por boto de prxima pgina

    df_all_search_results = pd.DataFrame(columns=["Title", "Sponsored", "Price"]) # cria dataframes de resultados
    
    # Procura pelo Boto Desabilitado de Prxima Pgina
    while not "s-pagination-disabled" in nextPageCssClasses:
        # Puxa as Informaes da Pgina
        soup = BeautifulSoup(wd.page_source)
        searchResultsPage = soup.findAll("div", {"data-component-type": "s-search-result"})
        df = getPandaDfForSearchResults(searchResultsPage)
        
        # Junta o Dataframe da Pgina com o Geral
        toConcat = [df for df in [df_all_search_results, df] if not df.empty]
        df_all_search_results = pd.concat(toConcat, ignore_index=True)

        # Vai para a Prxima Pgina
        nextPageLink.click()
        nextPageLink = wd.find_element(By.CLASS_NAME, "s-pagination-next")
        nextPageCssClasses = nextPageLink.get_attribute("class").split()
        time.sleep(2)
    
    # Termina Loop
    if "s-pagination-disabled" in nextPageCssClasses:
        print("Reached last page of search results")
    
    # Tratando Preos
    df_all_search_results['Price'] = df_all_search_results['Price'].str.replace(",","")
    df_all_search_results['Price'] = df_all_search_results['Price'].str.replace(".","")
    df_all_search_results['Price'] = df_all_search_results['Price'].astype(float)
    
    return df_all_search_results

def sendWhatsAppMessage(df):
    # Itera no Dataframe
    for index, graphicsCardRow in df.iterrows():
        # Acessa Link do Produto
        wd.get(graphicsCardRow.URL)
        # print(graphicsCardRow.URL)
        
        # Checa Existncia da Pasta e Cria
        if not os.path.exists(f'./Screenshot'):
            os.mkdir(f'./Screenshot')
        
        # Salva Screenshot da Pgina para Envio
        screenshotFilepath = f"./Screenshot/screenshot_{index}.png"
        wd.save_screenshot(screenshotFilepath)
        
        # Envia Mensagem no Whatsapp
        wpp.sendwhats_image(myPhoneNumber, screenshotFilepath, graphicsCardRow.URL, tab_close=True)

Conclusão:¶

A implementao de um tracker de preos no s beneficia consumidores, proporcionando economia e eficincia, como tambm oferece s empresas uma ferramenta poderosa para anlise de mercado e ajuste de estratgias. A metodologia descrita mostra-se eficaz para a coleta, anlise e notificao de preos, promovendo uma maior transparncia e competitividade no mercado. Esta anlise preliminar destaca a necessidade e os benefcios de um sistema automatizado de monitoramento de preos, incentivando a adoo de tecnologias que proporcionem eficincia e competitividade no mercado de consumo. Alm disso, com a anlise dos preos encontrados, podemos inferir que os preos dos celulares (utilizando a premissa de que so aqueles com valores acima de 4000 reais) possuem comportamentos parecidos, visto que so da mesma categoria. Ademais, os outliers encontrados so devido existncia de acessrios para ambos os produtos.
